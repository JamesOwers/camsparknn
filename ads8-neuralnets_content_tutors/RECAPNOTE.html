ADS8 - Neural Nets - Recap Note

In this short note we list the key-concepts that should be clear after the weekend as well as some pointers for extra-readings or references.

Key Concepts

The main "important" concepts that you should feel reasonably comfortable with essentially relate to day 1 (Multilayer Perceptron and Convolutional Neural Nets):

  • The perceptron and the link to a linear classifier
  • Trained parameters of a Neural Net: weights and biases
  • "Fixed" parameters (hyperparameters) of a Neural Net: architecture, activation functions, regularisation

A nice website to get your head around this is the tensorflow playground.

  • Convolution in 1D and 2D
  • Convolution layer
  • MaxPooling layer
  • Stochastic Gradient Descent
  • VGG16 as a simple "blackbox" feature extraction tool for images

More advanced concepts * Neural Network regularisation (Dropout, BatchNorm, weight regularisation, gradient clipping etc). There is a nice video tutorial on BatchNorm * Stepping schemes and initialisation (Adam, RMSProp, Glorot init, ...) * Recurrent Neural Nets (RNNs, LSTMs, GRU) * Issues with RNNs (vanishing gradients, slow training, lots of hyperparameters to tune) * ...

Walk-through examples

Here are some exercises that you could try doing if you wanted to get more of a grip of the Keras API and applications of NNs:

Image recognition: * Implement a simple ConvNet (with maybe ~100-1000 neurons) on fashionMNIST (benchmarks). To get higher scores you will need to tweak the regularisation parameters etc which is an excellent way to get a grip with Keras.

Sequences: * Finish the RNN-regression notebook. * Check the Kaggle-web-traffic problem. Top solutions are available on Github: here and here. Going through one of those and trying to reproduce the results is very likely to help you a lot with understanding how the API works.
